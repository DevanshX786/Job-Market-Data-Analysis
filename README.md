# Job Market Data Analysis

Project setup for extracting insights about skills demand, salary trends, and hiring patterns.

## Folders
- `data/raw`: Raw data files
- `data/processed`: Cleaned and processed datasets
- `src`: Python scripts for data analysis
- `docs`: Documentation and reports

## Setup
To set up the environment, simply activate the virtual environment and install the requirements:
```bash
# Windows
venv\Scripts\activate

# Install requirements
pip install -r requirements.txt
```
